### 0-1.Erlangのインストール

ElixirはErlangVM上で動作するので、当然Erlangが必要になります。 2014/8/25時点の最新バージョンは17.1です。

以下の手順でインストールします。

```bash
brew install erlang
```

### 0-2.rebarのインストール

rebarはErlangのビルドツールです。 Elixirのビルドに必要なので以下の手順でインストールして下さい。

```bash
brew install rebar
```

rebarの使い方は[こちら](https://github.com/rebar/rebar/wiki)を参照して下さい。
